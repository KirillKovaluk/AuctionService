Название проекта
AuctionService

Описание:
Веб-приложение для покупки и продажи на аукционе в соответствии с типом товара.

Структура программы:
1) База данных
2) Сервер
3) Веб-клиент

Преимущества:
1) Прогнозирование цен на лоты (на основе истории торгов, типов товара и ИИ)
2) Автоматическое выставление на торговую площадку продавцом
3) Автоматическая ставка покупателя

Функции:

Guest
Просмотр
Регистрация
Вход в аккаунт

User
+права Guest
Создание лота
Удаление лота
Редактирование
Выставление лота 
Участие в аукционе

+права User

Блокировка/разблокировка пользователя

System
